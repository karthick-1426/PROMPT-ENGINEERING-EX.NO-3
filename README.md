#  Introduction

Prompt Engineering is an important concept in Artificial Intelligence where instructions are carefully designed to obtain accurate and meaningful responses from AI systems.

Different prompting methods help AI understand the user's requirements more effectively and generate relevant responses.

In this experiment, an AI-powered customer support chatbot is used to demonstrate various prompting techniques in a practical and user-friendly manner.

### Chatbot Workflow

```text
Customer Query → AI Chatbot → AI Response
```

---

#  1. Straightforward Prompting

## Definition

Straightforward prompting gives direct instructions to the AI system without additional context.

### Example Prompt

```text
Track my order number ORD45678.
```

### AI Response

```text
Your order has been shipped and will arrive within 2 days.
```

### Advantages

* Simple and easy to understand
* Generates quick responses
* Suitable for customer support applications
* Reduces interaction time

---

#  2. Tabular Format Prompting

## Definition

Tabular prompting organizes information into rows and columns for easier readability and comparison.

### Example

| Customer Issue       | Chatbot Solution              |
| -------------------- | ----------------------------- |
| Order Delayed        | Track shipment details        |
| Password Reset       | Provide reset link            |
| Product Not Charging | Suggest troubleshooting steps |
| Refund Request       | Guide refund process          |

### Advantages

* Improves information organization
* Easy to compare multiple cases
* Better readability
* Useful for business reports

---

#  3. Preceding Question Prompting

## Definition

In this method, a related question is asked before the main question to provide context to the AI system.

### Example Prompt

```text
Why is customer support important?

Explain how AI chatbots improve customer service.
```

### AI Response

```text
Customer support improves customer satisfaction and trust.

AI chatbots improve customer service by providing instant responses, reducing waiting time, and offering 24/7 assistance.
```

### Advantages

* Improves contextual understanding
* Produces detailed responses
* Enhances answer quality
* Encourages reasoning

---

#  4. Missing Word Prompting

## Definition

Missing Word Prompting asks the AI system to complete an incomplete sentence.

### Example Prompt

```text
The order status is ______.
```

### AI Response

```text
The order status is delivered.
```

### Advantages

* Useful for learning systems
* Supports auto-completion
* Improves interaction with AI
* Helpful in educational applications
